Prueba - Backend Developer

Descripción del Proyecto

Esta aplicación tiene como objetivo cumplir con los requisitos de la prueba, que consiste en desarrollar una API con un CRUD de un catálogo de personas. Se aplicarán los principios de Clean Architecture y 12 Factor Apps.

Requisitos Clave:

  • Base de datos en MySQL.
  • Implementación de CRUD para un catálogo de personas.
  • Uso de patrones como CQRS y Mediator.
  • Implementación de Swagger para la API.
  • Interfaz básica para consumir la API.
  • Preparación para correr en contenedores.
  • Suite de pruebas con BDD o TDD.
  • Documentación en README.md.

Una vez completado, sube el código a un repositorio de GitHub y contacta al departamento de Recursos Humanos para agendar una reunión de presentación.
